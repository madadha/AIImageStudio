SmartChat.WPF

אפליקציית צ'אט חכמה שנבנתה באמצעות WPF בשפת #C, המאפשרת תקשורת עם מודלי בינה מלאכותית (Gemini ו-ChatGPT), כולל שימוש בכלים (Tools) חיצוניים ויצירת תמונות באמצעות AI.

🎯 מטרת הפרויקט

מטרת הפרויקט היא לבנות מערכת חכמה אשר:

מאפשרת תקשורת עם מודלי AI
תומכת בזיכרון שיחה
מאפשרת בחירת מודל
תומכת בהזרמת תשובות (Streaming)
משתמשת בכלים חיצוניים (Tools)
מאפשרת יצירת תמונות באמצעות בינה מלאכותית
🧠 טכנולוגיות בשימוש
WPF (Windows Presentation Foundation)
C#
Gemini API (Google)
OpenAI API
JSON Processing
